## How to Contribute
1. Fork the repository
2. Create a feature branch
3. Commit your changes
4. Push to the branch
5. Open a Pull Request

## Code of Conduct
- Be respectful
- Follow the existing code style
- Write meaningful commit messages
- Document your changes
